# Documentación de MesaUTP

Carpeta de documentación y diseño del proyecto, para todo el equipo.

## Archivos

- **`DOCUMENTACION_REQUERIMIENTOS.md`** — Requerimientos funcionales y no funcionales,
  actores, modelo de datos, endpoints de la API y stack. Actualizado con el modelo de
  autenticación vigente:
  - **Estudiante (consumidor):** enlace mágico al correo `@utp.edu.pe`.
  - **Emprendedor:** correo + contraseña (externo: cualquier correo; interno: `@utp.edu.pe`).

- **`pencil-new.pen`** — Prototipo de alta fidelidad de todas las pantallas (consumidor,
  emprendedor y admin). Es un archivo de **Pencil**; ábrelo con la app/editor de Pencil
  (no es texto plano).

## Repos del proyecto

- **Backend** (este repo): API REST en Laravel 12 + panel admin Filament en `/admin`.
- **Frontend**: PWA en React 19 + Vite.
